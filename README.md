# Amazon Clone

Frontend clone project inspired by the Amazon homepage, created to practice layout structure, styling and multi-section page composition.

## Technologies
- HTML5
- CSS3

## Main features
- Header and navigation bar layout
- Promotional sections and content blocks
- Card/grid-based product area
- Image-based sections and footer area
- Styling and spacing practice for a complex page

## Notes
This repository also includes a separate JavaScript exercise page (`JavaScript.html`) used during coursework practice.

## Project goal
Project created during the Front-End path at Epicode to improve HTML/CSS layout skills and build more structured, realistic UI pages.
